DetectorSleep es una aplicación web orientada al monitoreo del estado de alerta del usuario mediante visión por computadora. Integrada con modelos entrenados en Teachable Machine, la plataforma analiza en tiempo real la señal de la cámara web para determinar si el usuario se encuentra despierto o en estado de somnolencia. En caso de detectar inactividad prolongada o indicios de sueño, el sistema activa alertas automáticas con el objetivo de prevenir descuidos y mejorar la atención.
